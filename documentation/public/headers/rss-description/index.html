RSS-Description

The RSS-Description field can be set on the root node to set the <channel> <description> field in the generated RSS feed.

This will show up as a description of the web site in RSS readers.

The value of the RSS-Description header on the root node of this documentation web site is: 

Uriel Documentation

You can see how it shows up in the <channel> <description> portion of the generated RSS feed here: /rss.xml

This header is optional if you don't want to generate an RSS feed. However, if you do want to generate an RSS feed, then this header is required. If the RSS-URL header is set, then this header must also be set.

See the RSS, Sitemap, robots.txt page for additional details about how to generate an RSS feed.
