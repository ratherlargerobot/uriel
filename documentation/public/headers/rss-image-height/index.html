RSS-Image-Height

The RSS-Image-Height header can optionally be set on the root node to set the RSS <channel> <image> <height> field in the generated RSS feed.

This will specify the size of the image defined in the RSS-Image-URL header for RSS readers.

If the RSS-Image-URL header is not set, then this header will not have any effect.

The value of the RSS-Image-Height header on the root node of this documentation web site is: 

32

You can see how it shows up in the <channel> <image> <height> portion of the generated RSS feed here: /rss.xml

If you set this header, you should set the RSS-Image-Width header as well.
