Breadcrumb-Separator-Spaces

Breadcrumbs are a navigation element that shows where you are in the context of a web site. You can see them at the very top of this page.

Breadcrumb-Separator-Spaces controls whether spaces are added to either side of the Breadcrumb-Separator when breadcrumbs are generated using the {{breadcrumbs:*}} substitution parameter.

There are two valid values:

  • true
  • false

The default value is true

Here are two example pages highlighting the difference:

Breadcrumb-Separator-Spaces: true

Breadcrumb-Separator-Spaces: false
