{{breadcrumbs:*}}

Breadcrumbs are a navigation element that shows where you are in the context of a web site. You can see them at the very top of this page.

Uriel has two different substitution parameters to generate breadcrumbs.

  • {{breadcrumbs:*}}
  • {{breadcrumbs-root:*}}

The {{breadcrumbs:*}} parameter generates links to all of the pages leading up to the current node, but excludes the root node from the breadcrumb links.

This can be useful if the link to the root of the site gets special treatment (like an image link with a logo or something).

On this page of the documentation site, the {{breadcrumbs:*}} parameter evalutes to the following value:

Substitution Parameters » {{breadcrumbs:*}}

Compare this with the {{breadcrumbs-root:*}} parameter.
