Breadcrumb-Separator

Breadcrumbs are a navigation element that shows where you are in the context of a web site. You can see them at the very top of this page.

Breadcrumb-Separator sets the separator string used by the {{breadcrumbs:*}} substitution parameter.

The default value is &raquo; which looks like this when rendered in HTML: »

Notice how on this page, the navigation breadcrumbs at the top of the page use a slash as a separator, unlike every other page on this site? This is because the Breadcrumb-Separator header has been set to / on the content node for this page.

This header is closely related to the Breadcrumb-Separator-Spaces parameter, which controls whether spaces will be added on either side of the breadcrumb separator.
